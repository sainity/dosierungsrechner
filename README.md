# dosierungsrechner
Tablettenmenge nach Körpergewicht berechnen – für Mensch und Tier.

Hinweis für Tiere: Diese Berechnung ist eine rein mathematische Umrechnung. 
Dosierungen für Tiere weichen oft erheblich von menschlichen Dosierungen ab – viele Wirkstoffe sind für bestimmte Tiere (z. B. Katzen) giftig. Bitte konsultiere immer einen Tierarzt.
